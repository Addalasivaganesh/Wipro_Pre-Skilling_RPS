# Micro Machine Learning Projects

##  Description
This repository contains examples of Linear Regression models implemented using Python.

The main focus is:
- Simple Linear Regression (SLR)
- Multiple Linear Regression (MLR)

---

##  Concepts Covered

- Linear Regression
- Input (X) and Output (y)
- Train-Test Split
- Model Training using LinearRegression
- Model Evaluation (MAE, MSE, RMSE, R2 Score)

---

##  Project Structure

### 1. Marks Prediction
- Simple Linear Regression (StudyHours → Marks)
- Multiple Linear Regression (StudyHours + SleepHours → Marks)

### 2. Salary Prediction
- Simple Linear Regression (Experience → Salary)
- Multiple Linear Regression (Experience + Projects → Salary)

### 3. House Price Prediction
- Simple Linear Regression (Area → Price)
- Multiple Linear Regression (Area + Bedrooms + Age → Price)

### 4. Sales Prediction
- Simple Linear Regression (AdvertisingSpend → Sales)
- Multiple Linear Regression (AdvertisingSpend + ProductPrice + StoreSize → Sales)

---

##  Tools Used
- Python

##  Conclusion
This repository demonstrates the difference between:
